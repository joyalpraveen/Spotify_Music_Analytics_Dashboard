# Spotify Music Intelligence Dashboard

An interactive **Power BI dashboard** that analyzes Spotify music data to uncover insights about song popularity, artists, genres, and audio features. This project demonstrates data visualization, dashboard design, and business intelligence using Power BI.

---

## Project Overview

This dashboard provides an interactive view of Spotify music data through two pages:

### Page 1 – Popularity Dashboard
- Total Songs
- Total Artists
- Total Genres
- Average Popularity
- Top 10 Songs by Popularity (with Album Covers)
- Top 10 Artists by Average Popularity
- Average Popularity by Track Genre
- Genre & Artist Filters

### Page 2 – Audio Features Analysis
- Average Danceability
- Average Energy
- Average Valence
- Average Tempo
- Danceability vs Popularity (Scatter Plot)
- Top 10 Genres by Average Energy
- Average Tempo by Genre
- Top 10 Genres by Song Count (Treemap)
